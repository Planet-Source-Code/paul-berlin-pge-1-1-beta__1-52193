    PgeTiles
    PgeTiles is a little program used to make tilesets/sprites ready for use with Pab Game Engine. Pge is available on http://www.planetsourcecode.com as well as programs demonstrating the engine.

    You can create tilesets containing any number of tiles, which then can easily be accessed from your program using Pge with the correct class. These tiles can then be used as sprites in your program. The tiles are built using resources (which actually are simple PNG-Images.) containing one or more frames of the tile. The tiles can be semi-transparent and any number of frames and subframes can be used. Subframes are different images of the same frame, for example different angles of an object.
